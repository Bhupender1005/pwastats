## Developing

This site is built with [Jekyll](https://jekyllrb.com/docs/home/).

### Quick start

```sh
git clone git@github.com:cloudfour/pwastats.git
cd pwastats
gem install bundler
bundle install
npm install
```

For local development:

```
npm start
```

For a single build:

```
npm run build
```

View the local site at http://localhost:4000.
